# nonebot-plugin-asoul
  内置了asoul语音包的Nonebot2插件
## 📖 介绍
内置了嘉然和向晚的语音包，群消息发送指令即可
## 💿 安装
目前仅支持克隆到本地的方式
## 🎉 使用
### 指令表
| 指令 | 说明 |
|:-----:|:----:|
| 语音包 + 人物 | 发送人物随机语音包 |
| 语音包 + 人物 + 关键词 | 发送指定的语音包 |
