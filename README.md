# nonebot-plugin-asoul
  内置了语音包的asoul插件
## 📖 介绍
内置了嘉然的语音包，群消息发送指令即可
## 💿 安装
目前仅支持克隆到本地的方式
## 🎉 使用
### 指令表
| 指令 | 说明 |
|:-----:|:----:|
| 语音包 + 人物 + 关键词 | 发送匹配的语音包 |  

### 例子
语音包 嘉然 起床
